## Step-by-Step Method
1. **运球阶段（准备）**  
   - 动作：运球时提前屈髋屈膝降重心，保持低姿态。  
   - 观察点：重心是否在运球过程中持续压低，无抬高再下降。  
   - APP 检查点：检测髋关节角度变化曲线，是否在运球过程中保持稳定下降趋势。

2. **胯下/背后运球衔接**  
   - 动作：运球路径尽量从膝下穿过，球与身体重心在同一冠状面。  
   - 观察点：球是否从膝下穿过，而非臀下/臀外。  
   - APP 检查点：检测球轨迹与髋关节水平位置关系，判断是否在同一冠状面。

3. **触地瞬间**  
   - 动作：脚触地后快速衔接脚跟上抬、前脚掌蹬地，无二次屈膝蓄力。  
   - 观察点：触地后是否有明显停顿或再次下蹲。  
   - APP 检查点：检测触地时刻到蹬伸开始的时间间隔（<0.1秒为流畅）。

4. **合球与起球**  
   - 动作：合球接近身体，以肘为轴向上折叠小臂，蹬地同时起球。  
   - 观察点：球是否贴近身体，起球路径是否垂直向上。  
   - APP 检查点：检测肘关节角度变化与膝关节角度变化起始时间差（<0.05秒为协同）。

5. **发力与出手**  
   - 动作：下肢蹬伸完成瞬间衔接上肢伸展，从下肢到躯干到上肢充分伸展。  
   - 观察点：出手后重心是否稳定，手型是否正直。  
   - APP 检查点：检测出手瞬间肩、肘、腕是否在一条垂直线上；检测重心水平位移量。

## Repair Actions / Practice Tasks
1. **训练：低重心运球衔接投篮**  
   - drill: 原地低重心运球（髋关节<90°），保持重心稳定，然后做一次胯下运球后直接起跳投篮。  
   - purpose: 培养持续低重心习惯，消除重心起伏。  
   - setup: 无防守，半场，使用标准篮球。  
   - dosage: 10次/组，3组，组间休息1分钟。  
   - success_metric: 运球过程中髋关节高度变化<5cm，投篮命中率>60%。

2. **训练：蹬地同时起球节奏练习**  
   - drill: 无球状态下，站立屈膝，喊“蹬”时快速蹬伸下肢，同时模拟起球动作（肘部折叠）。  
   - purpose: 建立上下肢协同发力节奏。  
   - setup: 无球，镜子前或录像。  
   - dosage: 20次/组，3组。  
   - success_metric: 膝关节伸展起始与肘关节折叠起始时间差<0.05秒。

3. **训练：膝下运球路径训练**  
   - drill: 做胯下运球，要求球从膝关节下方穿过，保持身体重心低。  
   - purpose: 优化运球路径，使球与身体重心在同一冠状面。  
   - setup: 无防守，使用标志桶标记膝高位置。  
   - dosage: 20次/组，3组。  
   - success_metric: 球轨迹最低点低于膝高，且位于髋关节正下方。

4. **训练：合球后无陈球投篮**  
   - drill: 接球后直接起球投篮，禁止球下放。使用口令“接-起-投”。  
   - purpose: 消除二次蓄力习惯。  
   - setup: 队友或发球机传球。  
   - dosage: 15次/组，3组。  
   - success_metric: 合球后球无向下位移，投篮动作流畅。

## False Positives
- **重心起伏误判**：用户可能因疲劳或防守压力导致重心自然变化，不应直接判定为两段式习惯。需多次重复观察。
- **陈球动作误判**：部分一段式射手在接球后会有微小的“下沉”动作（<2cm），属于弹性蓄力，不应视为错误。需设定阈值（>5cm）。
- **运球路径误判**：用户身高臂展不同，膝下运球标准需个体化调整。高个子可能无法完全做到膝下，应允许适当调整。
- **发力时序误判**：用户可能因力量不足导致下肢滞后，而非习惯问题。需结合力量评估。
