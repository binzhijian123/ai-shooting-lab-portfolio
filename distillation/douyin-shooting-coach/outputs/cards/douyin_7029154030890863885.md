## Step-by-Step Method
1. **准备与站位**  
   - 动作：双脚采用轻微内八站位，左脚脚尖正对篮筐，右脚脚尖内旋，形成侧身。  
   - 观察点：双脚间距窄（窄距站位），身体侧对篮筐。  
   - APP 检查点：检测脚尖朝向角度、双脚间距、髋部与肩部朝向是否形成侧身。

2. **接球与沉球（下肢蓄力）**  
   - 动作：接球时利用球下落惯性顺势沉球，同时降重心、屈膝屈髋（幅度小），前脚掌着地蓄力。  
   - 观察点：沉球动作流畅无停顿，膝盖弯曲角度较小，重心下降。  
   - APP 检查点：检测沉球时机（是否与接球同步）、膝盖弯曲角度（是否小于传统投篮）、前脚掌是否先着地。

3. **起跳与发力**  
   - 动作：前脚掌快速蹬地，带动小腿肌肉收缩，触发牵张反射，快速有力起跳。  
   - 观察点：起跳速度快，无明显停顿，身体竖直向上。  
   - APP 检查点：检测起跳速度（帧间位移）、起跳方向（是否竖直）、脚掌离地顺序（前脚掌先离地）。

4. **起球与出手**  
   - 动作：利用起跳力量顺势起球，投篮手空手心持球，五指微张，快速压腕拨球，出手后手臂未完全伸直，中指指向地面。  
   - 观察点：起球与起跳同步，出手点高，手臂不完全伸直，手腕充分下压。  
   - APP 检查点：检测起球时机（是否与起跳同步）、出手后手臂伸直角度（是否小于180度）、手腕压腕角度、中指指向。

5. **辅助手动作**  
   - 动作：辅助手在投篮手开始拨球时脱离篮球，出手后上举，不向前跟随。  
   - 观察点：辅助手在出手前脱离，出手后保持上举。  
   - APP 检查点：检测辅助手脱离时机（是否在拨球前）、出手后辅助手位置（是否上举）。

6. **转宽送肩（远距离投篮时）**  
   - 动作：在三分或远距离投篮时，向左侧旋转髋部，右肩向前送，帮助身体纠偏和力量传导。  
   - 观察点：髋部旋转、右肩前送，身体形成更明显的侧身。  
   - APP 检查点：检测髋部旋转角度、肩部前送幅度、是否与投篮方向一致。

## Common Mistakes
1. **错误：沉球与接球不同步，出现停顿**  
   - symptom: 接球后先停顿再沉球，或沉球动作生硬。  
   - likely_cause: 未利用球下落惯性，主动发力沉球。  
   - evidence_needed: 视频中检测接球到沉球的时间间隔（应小于0.2秒）。

2. **错误：起跳速度慢，屈膝幅度过大**  
   - symptom: 膝盖弯曲角度大，起跳有明显下蹲停顿。  
   - likely_cause: 采用传统大肌群发力，未使用压弹式发力。  
   - evidence_needed: 检测膝盖弯曲角度（应小于传统投篮的90度）、起跳前停顿时间。

3. **错误：出手后手臂完全伸直**  
   - symptom: 出手后手臂完全伸直，手腕压腕不充分。  
   - likely_cause: 追求力量而非速度，未采用快速出手技术。  
   - evidence_needed: 检测出手后肘关节角度（应小于180度）、手腕压腕角度。

4. **错误：辅助手干扰投篮**  
   - symptom: 辅助手在出手后向前跟随或未及时脱离。  
   - likely_cause: 辅助手参与发力或未养成独立出球习惯。  
   - evidence_needed: 检测辅助手脱离时机（是否在拨球前）、出手后辅助手位置。

5. **错误：投篮挺腰，起跳不竖直**  
   - symptom: 起跳后腰部后仰，身体前倾或后仰。  
   - likely_cause: 核心未收紧，脊柱未保持中立位。  
   - evidence_needed: 检测起跳后脊柱角度（应与地面垂直）、腰部是否后仰。

## Repair Actions / Practice Tasks
1. **训练：接球沉球一体化练习**  
   - drill: 队友或机器传球，接球瞬间顺势沉球，无停顿。  
   - purpose: 培养利用球下落惯性沉球的习惯。  
   - setup: 距离篮筐5-6米，接球后不投篮，仅做沉球动作。  
   - dosage: 每组10次，共3组。  
   - success_metric: 接球到沉球时间<0.2秒，动作流畅无停顿。

2. **训练：窄距压弹式起跳练习**  
   - drill: 双脚窄距站立，前脚掌快速蹬地起跳，不投篮。  
   - purpose: 掌握压弹式发力，减少屈膝幅度。  
   - setup: 原地或行进间，起跳后保持竖直。  
   - dosage: 每组15次，共3组。  
   - success_metric: 起跳前膝盖弯曲角度<90度，起跳速度<0.3秒。

3. **训练：快速出手投篮练习**  
   - drill: 近距离（2-3米）投篮，强调快速压腕拨球，出手后手臂不完全伸直。  
   - purpose: 培养快速出手习惯。  
   - setup: 篮下或罚球线附近，不追求力量。  
   - dosage: 每组20次，共3组。  
   - success_metric: 出手后肘关节角度<170度，手腕充分压腕。

4. **训练：辅助手脱离练习**  
   - drill: 单手投篮（辅助手不接触球），或辅助手仅轻触球，出手前主动移开。  
   - purpose: 消除辅助手干扰。  
   - setup: 近距离投篮，辅助手放在身体一侧。  
   - dosage: 每组15次，共3组。  
   - success_metric: 出手后辅助手上举，不向前跟随。

5. **训练：炸球衔接投篮练习**  
   - drill: 运球一次（炸球）后顺势起球投篮。  
   - purpose: 培养投篮节奏，解决手快脚慢问题。  
   - setup: 三分线附近，炸球后立即投篮。  
   - dosage: 每组10次，共3组。  
   - success_metric: 炸球到出手时间<1秒，动作流畅无停顿。

## False Positives
- 用户屈膝幅度小但起跳速度慢：可能因力量不足或协调性差，而非采用压弹式发力。  
- 用户出手后手臂不完全伸直但手腕压腕不充分：可能因手臂力量不足，而非快速出手技术。  
- 用户辅助手上举但脱离时机晚：可能因习惯性依赖辅助手，而非独立出球。  
- 用户起跳竖直但腰部后仰：可能因核心力量不足，而非刻意保持中立位。  
- 该规则不适用于近距离开（篮下）或跳投高度极低的用户，因压弹式发力在近距离可能不必要。
