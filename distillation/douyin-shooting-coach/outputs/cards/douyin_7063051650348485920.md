定义和诊断投篮中“手快脚慢”（发力脱节）的问题，并纠正传统判断误区，提供正确的发力节奏。

## Applicable Scenarios
- User scenario: 用户投篮感觉无力、射程不足，或视频分析显示上下肢动作不协调。
- Shot type: 一段式 (one-motion) 和 1.5段式 (1.5-motion) 投篮。

- Skill level: 中级及以上（已掌握基本投篮动作，但存在发力效率问题）。

## Core Claims
1. **传统判断标准不准确**：仅凭“抬球点仍屈膝”来判断手快脚慢是不准确的。库里、汤普森在抬球点仍屈膝，但发力是自下而上有序叠加的。  
   - evidence: explicit
2. **真正的手快脚慢定义**：在投篮起重心的阶段，下肢还未完全伸展时，上肢就提前发力，导致下肢力量与上肢力量脱节，未叠加到篮球上。  
   - evidence: explicit
3. **问题仅出现在一段式和1.5段式投篮**：两段式投篮因起跳到最高点后才上肢发力，不存在手快脚慢问题。  
   - evidence: explicit
4. **两种常见错误模式**：  
   - 模式一：降重心同时起球，下肢与上肢同时伸展，伸膝与顶肘同步。  
   - 模式二：降重心与沉球同步，但起球阶段上肢主动发力，下肢伸展速度慢。  
   - evidence: explicit
5. **正确发力节奏**：接球→快速降重心→利用身体向上的弹性顺势举球至抬球点→顺势拨球出手。  
   - evidence: explicit
6. **快速降重心的作用**：触发身体后侧链肌肉的牵张反射，产生向上的力量，上肢举球是被这股力量顺势带动的，不需要刻意用力。  
   - evidence: explicit
7. **一段式与1.5段式的区别**：一段式在抬球点几乎没有停顿，射程更远更省力；1.5段式在抬球点有停顿瞄篮，抗干扰能力更强更稳定。  
   - evidence: explicit

## Step-by-Step Method
1. **接球与降重心**  
   - 动作：接球后快速降重心（屈膝、屈髋）。  
   - 观察点：降重心是否快速、充分，是否与沉球（ball dip）同步。  
   - APP 检查点：检测降重心开始时间与沉球开始时间是否一致；检测膝盖弯曲角度是否达到合理范围（约90-120度）。

2. **利用牵张反射向上**  
   - 动作：快速降重心后，利用身体向上的弹性顺势起跳。  
   - 观察点：下肢伸展是否连贯、有弹性，没有停顿。  
   - APP 检查点：检测从最低点到起跳的过渡时间（应<0.2秒）；检测髋部、膝盖、脚踝是否同步伸展。

3. **顺势举球**  
   - 动作：上肢举球是被下肢向上的力量顺势带动的，不需要刻意用力。  
   - 观察点：举球是否与下肢伸展同步，没有提前发力。  
   - APP 检查点：检测肘部开始上升的时间与膝盖开始伸展的时间是否匹配（应几乎同时或略晚）；检测手腕是否在举球过程中保持放松。

4. **顶肘与出手**  
   - 动作：先伸膝（下肢伸展），再顶肘（上肢发力）。  
   - 观察点：顶肘是否发生在下肢伸展至最高点或接近最高点时。  
   - APP 检查点：检测肘部完全伸展的时间与脚离地时间的关系（应接近或略晚）；检测手腕拨球是否在顶肘之后自然完成。

5. **顺势拨球**  
   - 动作：所有力量传导到手腕后，让球顺势流出。  
   - 观察点：出手是否自然放松，没有刻意发力。  
   - APP 检查点：检测手腕角度变化是否平滑；检测球离手时手指的指向。

## Common Mistakes
1. **错误模式一：降重心同时起球**  
   - symptom: 下肢与上肢同时伸展，伸膝与顶肘同步。  
   - likely_cause: 对发力顺序理解错误，试图用上肢力量主导投篮。  
   - evidence_needed: 视频中检测膝盖开始伸展的时间与肘部开始上升的时间是否几乎相同。

2. **错误模式二：起球阶段上肢主动发力**  
   - symptom: 沉球与降重心同步，但起球时上肢主动用力，下肢伸展速度慢。  
   - likely_cause: 上肢力量过强或习惯性提前举球。  
   - evidence_needed: 视频中检测肘部上升速度是否明显快于膝盖伸展速度；检测手腕是否在举球过程中过早紧张。

3. **发力脱节导致投篮无力**  
   - symptom: 投篮距离不足，感觉用不上力。  
   - likely_cause: 上下肢力量未叠加，仅靠上肢发力。  
   - evidence_needed: 视频中检测出手时身体是否仍处于上升阶段；检测球离手时肘部是否未完全伸展。

## Repair Actions / Practice Tasks
1. **分解节奏训练**  
   - drill: 沉球-降重心-停顿-起跳-出手。  
   - purpose: 打破“降重心同时起球”的错误习惯，建立正确的时序感。  
   - setup: 在罚球线位置，无防守。  
   - dosage: 每组10次，做3组，组间休息30秒。  
   - success_metric: 在停顿点，膝盖处于屈膝状态，球在腰部或以下，然后下肢先伸展再顶肘。

2. **被动举球训练**  
   - drill: 仅用下肢力量起跳，上肢完全放松，让球被“弹”起来。  
   - purpose: 感受“顺势举球”而不是主动发力。  
   - setup: 近距离（篮下），使用轻球或普通篮球。  
   - dosage: 每组15次，做2组，组间休息1分钟。  
   - success_metric: 球离手时感觉不到上肢用力，出手自然流畅。

3. **牵张反射触发训练**  
   - drill: 快速下蹲后立即起跳，不举球，感受弹性。  
   - purpose: 激活后侧链肌肉的牵张反射，提高下肢爆发力。  
   - setup: 无球，原地进行。  
   - dosage: 每组10次，做3组，组间休息30秒。  
   - success_metric: 起跳高度和速度明显提升，感觉有“弹射”感。

4. **视频自我诊断**  
   - drill: 录制侧面视角的投篮视频，逐帧分析膝盖和肘部的运动时序。  
   - purpose: 量化诊断是否存在手快脚慢。  
   - setup: 手机或摄像机，侧面角度。  
   - dosage: 每次训练后分析3-5次投篮。  
   - success_metric: 能够准确识别膝盖开始伸展与肘部开始上升的时间差（应>0.1秒）。

## False Positives
- **抬球点仍屈膝**：不应直接判定为手快脚慢。库里、汤普森在抬球点仍屈膝，但发力顺序正确。需结合后续动作（下肢伸展与顶肘顺序）判断。
- **上肢看起来发力**：如果上肢是被下肢力量顺势带动，看起来可能像主动发力。需通过时序分析区分“主动”与“被动”。
- **两段式投篮**：视频明确指出两段式投篮不存在手快脚慢问题，因此该规则不适用于两段式投篮者。
- **力量不足或疲劳**：用户可能因力量不足而无法完成快速降重心，导致动作变形，这可能是体能问题而非技术问题。
