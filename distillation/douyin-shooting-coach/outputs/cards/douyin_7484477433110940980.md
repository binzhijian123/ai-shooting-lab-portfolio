## Step-by-Step Method
1. **准备姿势**：双脚与肩同宽或略宽，屈膝降重心，球置于胸前或腰部附近（准备位置）。
   - 观察点：重心是否稳定，球是否在身体中线附近。
   - APP检查点：检测膝盖弯曲角度（建议90-120度），球初始高度（不应高于头部）。
2. **启动发力**：下肢开始蹬地，同时上肢开始举球（同步进行）。
   - 观察点：蹬地动作与举球动作是否同时启动，没有明显先后顺序。
   - APP检查点：检测膝盖伸展与手腕/肘部抬升的起始时间差（应<0.05秒）。
3. **蹬伸与举球同步**：下肢持续蹬伸，躯干向上并略向后倾斜（约5-10度），球持续上升至发力点（约肩部或头部高度）。
   - 观察点：躯干是否形成略向后倾角，球是否在上升过程中保持稳定。
   - APP检查点：检测躯干倾斜角度（应向后5-10度），球上升轨迹是否垂直或略向后。
4. **发力点释放**：当下肢完全蹬伸（脚尖离地或接近离地）时，上肢顺势完成投篮释放（手腕拨球）。
   - 观察点：上肢发力点是否与下肢蹬伸完成点重合，没有停顿。
   - APP检查点：检测下肢蹬伸完成时刻与上肢释放时刻的时间差（应<0.05秒），检测手腕拨球动作是否流畅。
5. **跟随动作**：投篮后手臂自然跟随，手腕放松，身体保持平衡。
   - 观察点：手臂是否完全伸展，手腕是否指向篮筐。
   - APP检查点：检测手臂伸展角度（应接近180度），检测落地时身体是否稳定。

   - drill: 原地站立，双手持球于胸前，同时开始屈膝和举球动作，重复10次。
   - purpose: 建立上下肢同步启动的肌肉记忆。

   - dosage: 每天3组，每组10次。
   - success_metric: 侧面录像显示举球与屈膝动作同时启动（时间差<0.05秒）。

2. **训练：顺势释放训练**

   - purpose: 消除上肢发力点停顿，实现顺势投篮。
   - setup: 近距离（篮下或3米处），使用标准球。

3. **训练：向后倾角训练**

4. **训练：计时投篮训练**

## False Positives
- **“先举球再蹬地”模式在接球后快速投篮中可能被误判**：接球后球已在较高位置，举球动作可能不明显，但蹬地启动仍可能滞后。应检测蹬地启动时间，而非举球启动时间。
- **“重心前倾”可能由其他原因导致**：如防守压迫、疲劳、力量不足等。应结合其他指标（如膝盖弯曲角度、躯干稳定性）综合判断。
- **“出手速度慢”可能由其他因素导致**：如力量不足、协调性差、心理紧张等。应先排除这些因素，再归因于发力模式。
- **视频中“塞尔维亚球员”案例可能不具代表性**：视频仅举一例，未说明该球员是否代表所有“先举球再蹬地”模式。应谨慎推广。
