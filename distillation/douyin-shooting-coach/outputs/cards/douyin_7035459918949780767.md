## Step-by-Step Method
1. **准备姿势（持球放松）**  
   - 动作：侧腕持球，手掌与篮球自然贴合，五指放松不紧绷，刻意留出空隙。  
   - 观察点：持球手是否僵硬、五指是否过度张开或紧绷。  
   - APP 检查点：检测持球手手指间距、手掌与球接触面积（是否全掌贴合）。

2. **沉球与起球（利用惯性）**  
   - 动作：利用篮球下落的惯性沉球，利用身体上升的弹性起球，上肢不主动发力。  
   - 观察点：沉球时手臂是否主动下压，起球时是否与蹬地同步。  
   - APP 检查点：检测沉球阶段手臂肌肉是否紧张（如肩部上提），起球与蹬地的时间差。

3. **蹬地举球（动力链启动）**  
   - 动作：蹬地发力同时竖直举球，举球轨迹贴近身体，曲线平缓。  
   - 观察点：举球是否竖直、是否远离身体、轨迹是否平缓。  
   - APP 检查点：检测举球轨迹的垂直度、与身体的距离（肩关节角度）、轨迹曲率。

4. **甩鞭传递（逐节甩动）**  
   - 动作：肩关节、肘关节、腕关节依次逐节甩动，保持柔性连接，上肢放松。  
   - 观察点：各关节是否依次启动、是否有停顿或僵硬。  
   - APP 检查点：检测肩、肘、腕关节的启动时序（如肘关节是否先于肩关节启动）。

5. **顺势压腕（出手控制）**  
   - 动作：快速压腕，无停顿，五指指向篮筐方向。  
   - 观察点：压腕是否流畅、手指方向是否一致。  
   - APP 检查点：检测压腕速度、手指指向与篮筐方向的偏差角。

## Common Mistakes
1. **错误：上肢主动推球（推投）**  
   - symptom: 投篮距离不足，球弧度低，手臂前伸明显。  
   - likely_cause: 用户习惯用肩、肘主动发力，动力链脱节。  
   - evidence_needed: 检测肩关节是否在蹬地前或同时主动前推，肘关节是否先于肩关节启动。

2. **错误：核心松垮导致动量损失**  
   - symptom: 投篮时身体晃动，球轨迹不稳定，力量传递中断。  
   - likely_cause: 核心肌群未收紧，躯干无法作为稳定的“鞭根”。  
   - evidence_needed: 检测躯干在投篮过程中的稳定性（如髋部是否旋转过度）。

3. **错误：举球远离身体**  
   - symptom: 球在出手前远离身体，身体后仰或失衡。  
   - likely_cause: 大臂前举，破坏身体平衡和力量传导。  
   - evidence_needed: 检测举球轨迹与身体的距离（肩关节外展角度）。

4. **错误：压腕停顿或五指紧绷**  
   - symptom: 出手后球旋转不足，方向不稳定。  
   - likely_cause: 手腕主动发力而非顺势压腕，五指僵硬。  
   - evidence_needed: 检测压腕动作是否流畅（有无停顿），手指是否在出手后指向篮筐。

## Repair Actions / Practice Tasks
1. **训练：惯性沉球与起球**  
   - drill: 站立持球，让球自由下落，利用下落惯性沉球，然后利用身体上升（微屈膝后蹬地）的弹性起球，不主动用手臂发力。  
   - purpose: 培养上肢放松、利用惯性的感觉。  
   - setup: 无防守，近距离（如罚球线）。  
   - dosage: 10次/组，3组。  
   - success_metric: 沉球和起球时手臂肌肉无明显紧张，球轨迹稳定。

2. **训练：竖直举球贴近身体**  
   - drill: 面对镜子或摄像头，从持球位置竖直举球至出手点，确保球轨迹贴近身体（大臂不远离躯干）。  
   - purpose: 纠正举球远离身体的错误。  
   - setup: 无球或轻球，侧面视角。  
   - dosage: 10次/组，3组。  
   - success_metric: 举球轨迹垂直，球与身体距离恒定（如肩关节外展角<30°）。

3. **训练：动力链逐节甩动（无球）**  
   - drill: 站立，模拟投篮动作，从踝→膝→髋→肩→肘→腕依次启动，强调“甩”的感觉，上肢完全放松。  
   - purpose: 建立动力链顺序和放松感。  
   - setup: 无球，侧面视角。  
   - dosage: 5次/组，3组。  
   - success_metric: 各关节启动有明显时序差，无同时启动或僵硬。

4. **训练：顺势压腕（近距离）**  
   - drill: 近距离（如篮下）投篮，专注于快速、流畅的压腕，五指指向篮筐，不主动发力。  
   - purpose: 培养顺势压腕的习惯。  
   - setup: 篮下，无防守。  
   - dosage: 10次/组，3组。  
   - success_metric: 出手后球旋转明显（至少2-3圈），方向稳定。

## False Positives
- **现象：用户投篮时手臂前伸**  
  可能误判为“推投”，但实际可能是用户为了增加距离而自然前伸，而非主动发力。需结合肩关节启动时序判断。

- **现象：用户投篮后手指指向篮筐**  
  可能误判为“正确压腕”，但实际可能是用户刻意控制手指方向，而非顺势压腕。需结合压腕流畅度判断。

- **现象：用户投篮弧度高**  
  可能误判为“甩鞭发力成功”，但实际可能是用户通过上肢主动发力（如高肘）实现。需结合动力链时序和上肢放松程度判断。

- **不适用场景**：  
  - 跳投高度大的投篮（如急停跳投），因为视频强调“站投三分”，动力链可能不同。  
  - 近距离投篮（如罚球），因为甩鞭发力可能不必要，且容易导致过度发力。
