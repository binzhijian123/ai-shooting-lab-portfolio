## Step-by-Step Method
1. **准备与降重心**
   - 动作: 接球或运球后，快速降重心，屈髋带动小幅屈膝，同时沉球至腰部附近。
   - 观察点: 重心下降是否与沉球同步；屈髋幅度是否大于屈膝。
   - APP 检查点: 检测髋关节和膝关节角度变化是否同步；检测沉球起始点与重心最低点的时间差。

2. **蓄力与蹬地**
   - 动作: 双脚前脚掌触地向下压，脚踝率先发力，前脚掌蹬地快速向上弹起。
   - 观察点: 脚跟是否轻微离地或前脚掌压地；蹬地是否迅速、无停顿。
   - APP 检查点: 检测脚踝角度变化（背屈到跖屈的转换速度）；检测前脚掌压力中心是否在蹬地瞬间前移。

3. **起球与上肢发力**
   - 动作: 下肢伸展的瞬间，上肢顺势顶肘压腕，手肘拉直的同时手腕下压。
   - 观察点: 起球是否跟随身体上升力量，无主动上举；顶肘和压腕是否流畅连续。
   - APP 检查点: 检测肘关节角度变化曲线是否平滑（无平台期）；检测手腕下压与肘关节伸展的时间差。

4. **转弯与拨球**
   - 动作: 侧腕持球与举球，在接近上肢发力点时，由侧腕转为正腕（约180度旋转），同时辅助手拇指参与拨球。
   - 观察点: 投篮手在举球过程中是否保持侧腕；转弯动作是否在发力点瞬间完成；辅助手拇指是否在出手时发力。
   - APP 检查点: 检测前臂旋转角度（从侧腕到正腕的旋转量）；检测辅助手拇指在出手瞬间的加速度。

## Common Mistakes
1. **错误：降重心与沉球不同步**
   - symptom: 先沉球再降重心，或先降重心再沉球，导致发力脱节。
   - likely_cause: 习惯性先屈膝再沉球，或沉球过晚。
   - evidence_needed: 视频中检测沉球起始点与重心最低点的时间差。

2. **错误：蹬地时脚跟离地过高或全脚掌着地**
   - symptom: 蹬地时脚跟完全离地（跳跃式）或全脚掌着地（无前脚掌压地）。
   - likely_cause: 脚踝发力不足或习惯性跳跃；或缺乏前脚掌压地意识。
   - evidence_needed: 检测脚踝角度变化和压力中心位置。

3. **错误：上肢发力有停顿或突然爆发**
   - symptom: 顶肘前有停顿，或手腕下压突然加速。
   - likely_cause: 上肢主动发力而非跟随身体上升力量；或发力点判断错误。
   - evidence_needed: 检测肘关节角度变化曲线是否有平台期；检测手腕加速度曲线是否平滑。

4. **错误：转弯时机错误或未转弯**
   - symptom: 举球过程中手腕始终正腕，或转弯发生在出手后。
   - likely_cause: 不熟悉侧腕持球；或转弯发力点判断错误。
   - evidence_needed: 检测前臂旋转角度变化与肘关节伸展的时间关系。

5. **错误：辅助手拇指过度发力**
   - symptom: 出手后篮球旋转异常（侧旋过多）或弹道偏移。
   - likely_cause: 辅助手拇指发力过大，干扰了投篮手拨球。
   - evidence_needed: 检测辅助手拇指在出手瞬间的加速度和方向。

## Repair Actions / Practice Tasks
1. **训练：沉球-降重心同步训练**
   - drill: 站立持球，同时做沉球（球降至腰部）和屈髋动作，确保动作同时开始和结束。
   - purpose: 建立降重心与沉球的同步节奏。
   - setup: 无球或轻球，站立姿势。
   - dosage: 3组，每组20次。
   - success_metric: 沉球起始点与重心最低点的时间差<0.05秒。

2. **训练：前脚掌蹬地专项训练**
   - drill: 原地站立，双脚前脚掌着地，快速向上弹起，强调脚踝爆发力。
   - purpose: 强化脚踝发力，提高蹬地效率。
   - setup: 无球，站立姿势。
   - dosage: 3组，每组15次。
   - success_metric: 脚踝背屈到跖屈转换时间<0.15秒。

3. **训练：跟随上升力量投篮训练**
   - drill: 从低位持球开始，下肢蹬地时顺势起球，不主动上举，只做顶肘压腕。
   - purpose: 培养上肢跟随身体上升力量的习惯，消除停顿。
   - setup: 近距离投篮（2-3米），轻球。
   - dosage: 3组，每组10次。
   - success_metric: 肘关节角度变化曲线无平台期，出手流畅。

4. **训练：侧腕持球-转弯练习**
   - drill: 站立无球，模拟侧腕持球，在发力点快速旋转手腕至正腕。
   - purpose: 掌握转弯时机和动作。
   - setup: 无球，站立姿势。
   - dosage: 3组，每组20次。
   - success_metric: 前臂旋转角度变化与肘关节伸展同步，无延迟。

## False Positives
- **误判：降重心与沉球不同步**——如果用户使用跳投或两段式投篮，降重心和沉球可能不同步，这是正常现象，不应套用此规则。
- **误判：蹬地时脚跟离地过高**——如果用户习惯性跳跃投篮（如跳投），脚跟离地是正常动作，不应视为错误。
- **误判：上肢发力有停顿**——如果用户使用两段式投篮，上肢发力点有停顿是技术特点，不应视为错误。
- **误判：转弯时机错误**——如果用户使用正腕持球（非侧腕），转弯技术不适用，不应套用此规则。
- **误判：辅助手拇指过度发力**——如果用户投篮弹道稳定，辅助手拇指发力可能是个人习惯，不应视为错误。
