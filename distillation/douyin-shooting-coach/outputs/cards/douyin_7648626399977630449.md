## Scoring Interpretation
This video exposes a partial scoring model, not a complete final formula.

- 74 分左右：上肢优先动，手先架起来；起球过早；髋被提前带伸展；力量压到膝盖；整体慢且不顺。
- 95 分左右：意识先想蹬地；手等脚；感到下肢力量后顺势摆球；脚带手；动作流畅。

Likely scoring axes:
- Timing score: 起球启动是否早于蹬地/下肢发力。
- Chain score: 下肢是否能带动上肢，而不是上肢牵着身体走。
- Joint-load score: 慢和卡顿是否集中在膝盖，而不是通过脚踝、膝盖、髋连续向上伸展。
- Fluency score: 从下沉、蹬地、摆臂、起球到伸展是否无明显停顿。
- Intent score: 球员的主观控制口令是否从“手先架起来”改成“先蹬地，手等脚”。

## Step-by-Step Method
1. No-ball jump check
   - Drill: 原地纵跳。
   - Cue: 蹬地、摆臂、手起来。
   - Observation: 身体是否由下肢把人向上送，而不是上肢先抢。
2. Ball-in-hand timing reset
   - Drill: 拿球后先屈膝屈髋沉下来。
   - Cue: 手放松，不要提前往前举，不要提前看篮筐。
   - Observation: 球是否在蹬地力量出现后才顺势上摆。
3. Full movement
   - Drill: 脚踝、膝盖、髋向上伸展后，手再动。
   - Cue: 不是手带脚，是脚带手。
   - Observation: 起球与身体上送合在同一条连续链里。

## Observable Movement Indicators
- Hip: 髋是否被上肢提前带伸展；是否随下肢蹬伸自然打开。
- Knee: 慢和卡顿是否集中在膝盖；力量是否过度压到膝盖。
- Ankle/foot: 是否先蹬地；脚是否承担启动节奏。
- Arms/ball: 手是否提前架球；球是否在下肢力量出现后顺势摆起。
- Timing: 手启动 vs 蹬地启动的先后关系。
- Fluency: 从沉到伸展到起球是否一气呵成。

## IF/THEN Diagnosis Rules
- IF: 上肢先动、手先架球，且下肢随后才启动
  THEN: 诊断为上肢抢先 / 手快脚慢，动作评分应下调。
  CHECK: 侧面视频中起球起始帧是否早于蹬地/身体上送起始帧。
  REPAIR: 原地纵跳找“蹬地-摆臂-手起来”，再带球复制。
  confidence_basis: explicit

- IF: 髋很早被带着伸展，力量更多压在膝盖
  THEN: 诊断为发力链被上肢提前牵动，膝盖成为慢点。
  CHECK: 髋伸展是否早于有效蹬地，膝盖是否长时间承压/停顿。
  REPAIR: 拿球后先屈膝屈髋沉下，手放松，等脚发力后顺势起球。
  confidence_basis: explicit plus biomechanics inference

- IF: 球员意识口令是“先蹬地，让手等脚”，且起球跟随下肢力量
  THEN: 判为更高效、更流畅的动作模式，可提高动作评分。
  CHECK: 蹬地、摆臂、起球是否连续。
  REPAIR: 保持脚带手，不追求第一时间举球。
  confidence_basis: explicit

## Repair Actions / Practice Tasks
1. 原地纵跳顺序练习
   - purpose: 建立下肢带动摆臂的基础节奏。
   - dosage: 3 组 x 8-10 次。
   - success_metric: 口令能稳定复述并执行“蹬地、摆臂、手起来”。
2. 持球手等待练习
   - purpose: 抑制上肢提前举球。
   - dosage: 3 组 x 8 次，近筐或无框均可。
   - success_metric: 手在蹬地力量出现后才起球，视频中无明显抢先。
3. 屈膝屈髋沉下再伸展
   - purpose: 找到脚踝、膝盖、髋向上伸展的连续链。
   - dosage: 3 组 x 6-8 次。
   - success_metric: 不把力量卡在膝盖，整体向上伸展更顺。

## False Positives / Boundaries
- 这条视频只说明 74 分与 95 分的动作差异，没有公开完整评分权重。
- 不应把“手等脚”理解成手故意慢到卡顿；目标是让手跟随下肢力量顺势起来。
- 不应把命中率当作动作评分唯一依据；视频明确在讲动作评分和发力流畅性。
- 若用户力量不足、伤病、视角遮挡或帧率过低，不能强行给总分。
