## Core Claims
1. **侧身侧腕是正身正腕整体左旋的结果**：侧身侧腕并非独立技术，而是在正身正腕基础上，下肢、躯干、手部整体向左旋转（右手球员）。  
   - evidence: explicit  
2. **肘正腕侧，腕正肘翻**：当手肘指向篮筐时，手腕自然处于侧位；若强行让手腕正对篮筐，手肘会外翻。  
   - evidence: explicit  
3. **起球过程中，小臂/手肘应对框，而非手腕过早对框**：手腕过早对框会切断下肢力量传导，导致单纯用手腕发力。  
   - evidence: explicit  
4. **转腕是全身旋转的结果，不是孤立动作**：手腕的转正是从脚下到躯干到小臂整体小幅旋转发力的结果，模仿手腕转动会导致发力不流畅。  
   - evidence: explicit  
5. **侧身站位提升下肢稳定性和弹性爆发**：侧身模式能让下肢基础更稳定，更好发挥弹性爆发。  
   - evidence: explicit  
6. **侧腕持球让肩肘更放松自然**：侧腕起球时，肩关节和肘关节能沿自然轨迹上抬。  
   - evidence: explicit  
7. **单纯用手腕投篮会限制射程和流畅性**：依赖手腕发力的球员，整体发力流畅性和射程受限。  
   - evidence: explicit  

## Step-by-Step Method
1. **准备站位**  
   - 动作：采用侧身站位（右手球员整体左旋），脚尖指向侧前方。  
   - 观察点：下肢、躯干、手部是否整体旋转，而非仅转脚或转手。  
   - APP检查点：检测躯干与下肢的旋转角度是否一致（如髋与肩的朝向差）。  

2. **持球与起球**  
   - 动作：侧腕持球，从偏右侧起球。  
   - 观察点：起球过程中，手肘指向篮筐，手腕保持侧位（不主动转正）。  
   - APP检查点：检测肘关节指向与篮筐方向夹角（应<15°），手腕与肘的相对角度（肘正时腕应侧）。  

3. **发力链条**  
   - 动作：从脚下开始，通过躯干、小臂的整体小幅旋转，带动手腕自然转正。  
   - 观察点：手腕转正发生在发力末端，是全身旋转的结果，而非主动转动。  
   - APP检查点：检测手腕转正的时间点（应在发力末端），以及髋、肩、肘的旋转时序（应从下到上）。  

4. **出手**  
   - 动作：利用下肢和肩肘力量投篮，手腕在最后自然拨球。  
   - 观察点：手腕不应过早对框，否则会切断下肢力量。  
   - APP检查点：检测手腕对框时刻与下肢发力时刻的同步性（手腕对框应晚于下肢伸展）。

## Common Mistakes
1. **错误：手腕过早对框**  
   - symptom: 起球时手腕主动转正，手肘外翻。  
   - likely_cause: 误解“手腕对框”为主动动作，而非全身旋转结果。  
   - evidence_needed: 视频中手腕对框时刻早于下肢伸展；肘关节外翻角度>15°。  

2. **错误：孤立转动手腕**  
   - symptom: 手腕在出手前主动转动，但下肢和躯干无旋转。  
   - likely_cause: 只模仿结果（手腕转正），未理解全身旋转链条。  
   - evidence_needed: 髋、肩、肘旋转角度接近0°，仅手腕有旋转。  

3. **错误：侧身但未整体旋转**

## False Positives
- **手腕在出手前自然转正**：如果用户是正身正腕投篮，手腕转正是正常的，不应被误判为错误。本规则仅适用于侧身侧腕投篮。  
- **手肘外翻但手腕正**：在某些投篮姿势（如正身正腕）中，手肘外翻可能是正常的。本规则不适用于正身正腕投篮。  
- **手腕过早对框但下肢力量强**：部分球员（如视频提到的“一些球员”）可能依赖手腕发力且射程不受限，此时不应套用本规则。
