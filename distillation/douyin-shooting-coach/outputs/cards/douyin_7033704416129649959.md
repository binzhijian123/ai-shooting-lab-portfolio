## Step-by-Step Method

2. **降重心（放松）**

3. **蓄力阶段（放松）**  
   - 动作：脚下放松，保持弹性。  
   - 观察点：脚掌是否贴地，有无提前蹬地迹象。

4. **发力起跳（瞬间收缩）**  
   - 动作：前脚掌快速蹬地，深吸（吸气）、深髋（伸髋），保持核心紧张。  
   - 观察点：蹬地是否有力，髋部是否伸展，核心是否稳定。  
   - APP 检查点：测量蹬地力（通过加速度或帧间位移），髋关节伸展速度，核心稳定性（躯干晃动幅度）。

5. **举球与出手（连贯传导）**  
   - 动作：双手顺势举球至头顶，身体充分伸展后顶肘、压腕、拨球。  
   - 观察点：举球是否流畅，肘部是否外展，手腕是否放松。  
   - APP 检查点：检测举球轨迹（是否垂直），肘部角度（是否在90度左右），手腕拨球动作（是否突然）。

## Common Mistakes

   - likely_cause: 肌肉紧张导致力量传导效率低。

2. **错误：降重心时主动撅屁股、膝盖过度前屈**  
   - symptom: 下蹲动作不自然，起跳无力。  
   - likely_cause: 错误地收紧核心和向后撅屁股，导致髋部紧张。  
   - evidence_needed: 视频中髋关节角度突然增大（后撅），膝盖弯曲超过90度或快速前移。

3. **错误：发力过早（起跳前已发力）**  
   - symptom: 投篮速度慢，球轨迹低。  
   - likely_cause: 在降重心或蓄力阶段就开始收缩肌肉。

## Repair Actions / Practice Tasks
1. **放松-发力切换训练**  
   - drill: 原地站立，先全身放松（手臂自然下垂，髋部放松），然后瞬间快速蹬地、伸髋、跳起。  
   - purpose: 训练从放松到发力的瞬间切换。  
   - setup: 无球，站在平坦地面。  
   - dosage: 每组10次，做3组，组间休息30秒。  
   - success_metric: 起跳前无预兆动作（如手臂提前上举），起跳速度快。

2. **自然下蹲训练**  
   - drill: 无球，模仿库里降重心动作：髋部自然下坐，膝盖小幅弯曲，保持躯干直立。  
   - purpose: 纠正主动撅屁股和膝盖过度前屈。  
   - setup: 面对镜子或摄像头，侧面视角。  
   - dosage: 每组15次，做3组，组间休息30秒。  
   - success_metric: 髋关节角度平滑下降，膝盖弯曲不超过90度，无后撅动作。

3. **延迟发力训练**  
   - drill: 持球，先做降重心动作（放松），在脚掌即将离地时才开始举球和出手。  
   - purpose: 延迟发力时机，避免起跳前发力。  
   - setup: 近距离投篮（如罚球线内）。  
   - dosage: 每组10次，做3组，组间休息1分钟。  
   - success_metric: 举球动作与起跳同步，球轨迹高且旋转强。

4. **弹性放松训练**  
   - drill: 无球，做快速下蹲-起跳动作，强调下蹲时保持弹性（像弹簧压缩）。  
   - purpose: 培养弹性放松，避免松垮。  
   - setup: 站在软垫上（增加反馈）。  
   - dosage: 每组10次，做3组，组间休息30秒。  
   - success_metric: 下蹲后立即起跳，无停顿，身体稳定。

## False Positives
- **误判：用户起跳前手臂轻微晃动被误判为紧张**  
  - 解释：放松状态下手臂可能自然晃动，但若晃动幅度大且伴随肌肉抖动，才可能是紧张。  
  - 建议：结合肌肉张力检测（如用户反馈）或多次重复动作分析。

- **误判：用户因力量不足导致球投不远，被误判为紧张**  
  - 解释：力量不足也可能导致球投不远，但紧张通常伴随动作僵硬。  
  - 建议：检查用户是否同时有旋转弱、动作流畅度差等特征。

- **误判：用户使用正腕持球但投篮效果好，被误判为错误**  
  - 解释：正腕持球不一定导致紧张，部分用户可能适应。  
  - 建议：仅当正腕持球伴随手腕或小臂紧张时才提示。

- **误判：用户降重心时膝盖弯曲较大但无紧张，被误判为错误**  
  - 解释：膝盖弯曲角度因人而异，需结合髋部是否后撅判断。  
  - 建议：仅当膝盖弯曲伴随髋部后撅或动作突然时才提示。
