## Topic
如何根据发力模式（顺势投篮 vs. 肌肉拉伸投篮）选择投篮时的双脚站距（窄距、标准、宽距）。

## Applicable Scenarios
- User scenario: 用户投篮发力不流畅、起跳慢、或下肢稳定性不足。
- Shot type: 跳投（一段式、两段式）。

## Core Claims
1. 窄距站位（如库里、安东尼）适用于顺势投篮（一段式），利用筋膜弹性和牵张反射快速起跳，更省力、更快。  
   - evidence: explicit
2. 标准站距（与肩同宽）适用于充分屈膝屈髋、利用肌肉拉伸力量投篮（两段式），利于下肢发力与平衡。  
   - evidence: explicit
3. 宽距站位（如汤普森）用于提高下肢稳定性，但会损耗部分下肢力量。  
   - evidence: explicit
4. 窄距起跳能提高下肢伸展速度，改善投篮。  
   - evidence: explicit
5. 标准/宽距站位稳定性更高，利于充分发力，常见于两段式投篮。  
   - evidence: explicit
6. 顺势投篮的屈膝屈髋幅度小，肌肉拉伸力量小，依赖筋膜弹性。  
   - evidence: explicit
7. 牵张反射产生的力量大于肌肉拉伸力量。  
   - evidence: explicit

## Step-by-Step Method
1. **确定发力模式**  
   - 动作：观察用户投篮时的屈膝屈髋幅度和起跳节奏。  
   - 观察点：屈膝幅度小、起跳快（一二步或垫步）→ 顺势投篮（一段式）；屈膝幅度大、起跳慢 → 肌肉拉伸投篮（两段式）。  
   - APP 检查点：测量屈膝角度（髋、膝、踝）和起跳时间（从屈膝到离地）。

2. **选择站距**  
   - 动作：根据发力模式调整双脚站距。  
   - 观察点：顺势投篮 → 窄距（小于肩宽）；肌肉拉伸投篮 → 标准（与肩同宽）或宽距（大于肩宽）。  
   - APP 检查点：测量双脚间距（以肩宽为基准），标记为窄/标准/宽。

3. **验证效果**  
   - 动作：进行多次投篮，检查起跳速度和稳定性。  
   - 观察点：窄距应提高起跳速度；标准/宽距应提高稳定性。  
   - APP 检查点：分析起跳速度（帧数）和投篮后身体晃动幅度。

## False Positives
- 用户因疲劳或伤病导致屈膝幅度小，可能被误判为“顺势投篮”，实际应检查身体状态。
- 用户身高或腿长比例特殊，导致“窄距”或“宽距”的绝对距离与常人不同，应使用相对比例（与肩宽比较）。
