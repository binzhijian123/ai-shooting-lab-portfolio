## Step-by-Step Method
1. **建立髋关节后座蓄力主导的发力模型**
   - 动作：从弯腰降重心改为髋关节后座（类似坐椅子），带动小幅度自然屈膝，胸口朝向篮筐，腰背挺直。
   - 观察点：重心是否从脚尖前移回脚掌中部；膝盖是否过度前伸；腰背是否弯曲。
   - APP 检查点：检测髋关节角度变化（后座时髋角减小）；检测膝盖角度（屈膝幅度是否自然）；检测躯干倾斜角度（是否保持直立）。

2. **调整发力节奏为“沉球与降重心同步，蹬地同时起球”**
   - 动作：沉球（球下放至腰部或更低）与降重心（髋后座+屈膝）同步进行；蹬地瞬间开始起球，起球与下肢伸展同步。
   - 观察点：沉球和降重心是否同时开始；起球是否在蹬地后立即开始；球上升速度与下肢伸展速度是否匹配。
   - APP 检查点：检测沉球开始时间与降重心开始时间的时间差（应接近0）；检测起球开始时间与蹬地开始时间的时间差（应接近0）；检测球上升速度曲线与下肢伸展速度曲线的同步性。

3. **等待下肢完全伸展后上肢再发力出手**
   - 动作：下肢完全伸展（膝盖、髋关节伸直）后，上肢（肩、肘、腕）再发力将球推出，实现“充分向上后再向前”。
   - 观察点：球是否在下肢完全伸展前就已出手（手快脚慢）；出手点高度是否足够。
   - APP 检查点：检测下肢完全伸展的时刻与球出手时刻的时间差（应接近0或略晚）；检测出手时肩关节高度（是否高于或等于下肢完全伸展时的高度）。

4. **优化收球动作以促进协同发力**
   - 动作：在起球过程中，上肢向身体内收球（即球靠近身体中线），抑制大臂过早打开（大臂与身体夹角过大）。
   - 观察点：大臂与身体夹角是否在起球过程中保持较小；球是否远离身体。
   - APP 检查点：检测起球过程中大臂与躯干的夹角变化（应保持较小，如<45°）；检测球与身体中心的水平距离（应保持较近）。

## Common Mistakes
1. **错误：手快脚慢（下肢未完全伸展就出手）**
   - symptom: 出手点低，力量传导不畅，投篮距离不足。
   - likely_cause: 降重心与起球同步（而非沉球与降重心同步），导致上肢发力过早。
   - evidence_needed: 检测下肢完全伸展时刻与球出手时刻的时间差（应接近0或略晚）。

2. **错误：重心前倾（弯腰降重心）**
   - symptom: 投篮时身体前冲，稳定性差，容易受伤（膝关节、腰部）。
   - likely_cause: 未建立髋关节后座蓄力主导，依赖屈膝和弯腰。
   - evidence_needed: 检测躯干倾斜角度（是否保持直立）；检测髋关节角度变化（是否后座）。

3. **错误：大臂过早打开（收球不充分）**
   - symptom: 球远离身体，上下肢脱节，力量传导中断。
   - likely_cause: 收球动作不到位，未抑制大臂与身体夹角过大。
   - evidence_needed: 检测起球过程中大臂与躯干的夹角变化（是否保持较小）。

## Repair Actions / Practice Tasks

   - drill: 无球状态下，站立，模拟坐椅子动作，髋关节向后移动，带动小幅度屈膝，保持腰背挺直。重复10次。

   - dosage: 每天3组，每组10次。
   - success_metric: 能自然完成髋后座动作，膝盖不超过脚尖，躯干保持直立。

2. **训练：一二步急停+跳步衔接蓄力**
   - drill: 运球后做一二步急停，或后撤/侧撤跳步，落地时自然进入髋关节后座蓄力姿势。
   - purpose: 通过移动来锁定膝关节，调动身体后侧链，帮助找到髋关节发力感。
   - setup: 篮球场，持球。
   - dosage: 每天5组，每组5次。
   - success_metric: 急停或跳步后能稳定进入蓄力姿势，无重心前倾。

3. **训练：节奏分解练习（沉-蹬-起-出手）**
   - drill: 持球，先沉球与降重心同步（沉），然后蹬地同时起球（蹬+起），等待下肢完全伸展后出手（出手）。使用口令“沉-蹬-起-出手”控制节奏。
   - purpose: 纠正“手快脚慢”，建立正确的发力节奏。
   - setup: 篮球场，近距离投篮（如罚球线内）。
   - dosage: 每天5组，每组10次。
   - success_metric: 能稳定完成节奏，出手时下肢已完全伸展，球出手点高。

4. **训练：内收起球练习**
   - drill: 持球，起球过程中主动将球向身体中线内收，抑制大臂打开。可先无球模拟，再持球练习。
   - purpose: 优化收球动作，促进上下肢协同发力。
   - setup: 平地，持球。
   - dosage: 每天3组，每组10次。
   - success_metric: 起球过程中大臂与躯干夹角保持<45°，球靠近身体中心。

## False Positives

- 用户投篮时下肢完全伸展但球出手过早，可能由其他原因（如肩部力量不足）导致，需进一步诊断。
