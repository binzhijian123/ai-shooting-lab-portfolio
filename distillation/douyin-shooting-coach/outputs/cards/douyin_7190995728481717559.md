## Step-by-Step Method
1. **准备蓄力**  
   - 动作：双脚站立，屈膝下蹲，重心降低。  
   - 观察点：是否用整个脚掌支撑，还是抬起脚跟用足弓支撑。  
   - APP 检查点：检测脚跟是否离地；检测足弓区域是否明显拱起（侧面视角）。

2. **足弓支撑**  
   - 动作：蓄力时抬起脚跟，用足弓支撑身体重量。  
   - 观察点：足弓是否被压低后快速回弹（弹性反应）。  
   - APP 检查点：测量足弓高度变化（从初始到蓄力最低点）；检测回弹速度（帧间变化）。

3. **脚趾主动蹬地**  
   - 动作：在足弓回弹瞬间，脚趾主动抠地，向下蹬地，形成杠杆撬动身体向上。  
   - 观察点：脚趾是否明显弯曲、抓地；蹬地是否主动而非被动。  
   - APP 检查点：检测脚趾关节角度变化（屈曲程度）；检测蹬地时序（是否在足弓回弹后立即发生）。

4. **顺势起跳**  
   - 动作：蹬地后顺势伸展髋、膝、踝，向上起跳。  
   - 观察点：下肢伸展是否连贯、无停顿；上下肢是否同步。  
   - APP 检查点：检测髋、膝、踝伸展时序（是否从下到上流畅）；检测手部举球与下肢伸展的同步性（手快脚慢偏差）。

## Common Mistakes
1. **错误：整个脚掌支撑蓄力**  
   - symptom: 蓄力时脚跟不离地，足弓未参与支撑。  
   - likely_cause: 足弓功能弱（扁平足/高弓足）或习惯性错误。  
   - evidence_needed: 侧面视频检测脚跟是否离地；足弓高度变化。

2. **错误：被动蹬地（持续屈膝蓄力后被动伸展）**  
   - symptom: 双脚触地后无积极蹬地动作，踝关节伸展慢，手快脚慢。

3. **错误：脚趾未主动发力**

## Repair Actions / Practice Tasks
1. **足弓支撑练习**  
   - drill: 原地站立，抬起脚跟，用足弓支撑身体，保持平衡。  
   - purpose: 培养足弓支撑意识，激活足底肌群。  
   - setup: 赤脚或穿薄底鞋，站在平坦地面。  
   - dosage: 每次保持10-15秒，重复5组。  
   - success_metric: 能稳定抬起脚跟，足弓明显拱起，无晃动。

2. **脚趾抠地启动练习**  
   - drill: 原地微屈膝，脚趾主动抠地，然后快速向上跳起（不举球）。  
   - purpose: 训练脚趾主动发力与蹬地衔接。  
   - setup: 站立姿势，双手叉腰或自然下垂。  
   - dosage: 每组10次，重复3组，注意节奏（抠地→立即跳）。  
   - success_metric: 跳起时脚趾明显屈曲，起跳迅速无延迟。

3. **完整投篮脚下发力练习**  
   - drill: 持球做投篮蓄力动作，专注脚下：足弓支撑→脚趾抠地→蹬地起跳→出手（可先不投，只做动作）。  
   - purpose: 将脚下发力整合到投篮动作中。  
   - setup: 靠近篮筐，轻投或只做动作。  
   - dosage: 每组5次，重复5组，慢速练习。  
   - success_metric: 起跳时无手快脚慢，下肢伸展流畅，脚趾主动抠地。

## False Positives
- **脚跟离地不等于足弓支撑**：有些人脚跟离地但足弓塌陷（如扁平足代偿），需检测足弓高度。  
- **脚趾屈曲不等于主动发力**：脚趾可能因紧张而被动弯曲，需结合蹬地时序判断（是否在足弓回弹后立即发生）。  
- **手快脚慢可能由其他原因导致**：如上肢发力过早、核心不稳等，需排除其他因素。  
- **该规则不适用于所有投篮类型**：如原地罚球（无起跳）或低手投篮（脚下发力模式不同）。  
- **足弓功能异常者需先训练，不可直接套用**：扁平足/高弓足者强行足弓支撑可能导致不适或受伤。
