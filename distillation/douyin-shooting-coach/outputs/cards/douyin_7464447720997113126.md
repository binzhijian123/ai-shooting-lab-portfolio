## Step-by-Step Method
1.  **空中折叠（准备阶段）**
    - **动作**: 在接球或运球结束前，双脚同时离地，在空中完成髋、膝、踝三关节的轻微折叠（预压缩）。
    - **观察点**: 观察运动员是否在空中就开始屈髋、屈膝、屈踝，而不是落地后才开始下蹲。
    - **APP 检查点**: 检测运动员双脚离地瞬间的髋、膝、踝角度。理想状态是已经开始弯曲，而不是完全伸直。

2.  **触地反弹（发力阶段）**
    - **动作**: 双脚同时落地，利用身体自重和肌肉的千张反射（牵张反射）及筋膜弹性，快速从地面弹起。避免深度下蹲蓄力。
    - **观察点**: 观察落地后髋、膝、踝的压缩幅度是否过大。理想状态是“浅压缩”，快速反弹。
    - **APP 检查点**: 测量落地瞬间到最低点（最大屈曲角度）的髋、膝、踝角度变化。变化幅度小（浅压缩）为佳。同时测量从落地到再次离地的时间，时间短（触地即弹）为佳。

3.  **同步起球（衔接阶段）**
    - **动作**: 在双脚蹬地起跳的同时，将球从持球位置举到出手点。
    - **观察点**: 观察“蹬地”和“起球”两个动作是否同时开始，没有先后顺序。
    - **APP 检查点**: 检测蹬地发力（地面反作用力开始增加）的时刻与球开始向上移动的时刻是否同步。时间差越小，协同性越好。

4.  **稳定制动（落地阶段）**
    - **动作**: 双脚同时触地完成制动，利用臀部力量稳定身体，为下一次动作或投篮出手提供稳定基础。
    - **观察点**: 观察双脚是否同时落地，落地后身体是否晃动。
    - **APP 检查点**: 检测双脚触地时间差（理想为0）。检测落地后身体重心在水平方向上的偏移量。

## Common Mistakes
1.  **错误：深度下蹲蓄力**
    - **symptom**: 垫步落地后，髋、膝、踝大幅度弯曲，像坐椅子一样，起跳慢。
    - **likely_cause**: 习惯性依赖下肢力量蓄力，未利用弹性。
    - **evidence_needed**: 视频分析显示落地后髋、膝、踝角度变化过大（例如，膝盖弯曲超过90度），且从落地到起跳的时间较长。

2.  **错误：上下肢发力脱节**
    - **symptom**: 脚已经蹬地起跳了，球才开始向上举；或者球已经举到头顶了，脚才开始蹬地。
    - **likely_cause**: 节奏感差，未养成“蹬地与起球同步”的习惯。
    - **evidence_needed**: 视频分析显示蹬地发力时刻与球开始向上移动的时刻有明显的时间差。

3.  **错误：双脚不同时落地**
    - **symptom**: 垫步后一只脚先落地，另一只脚后落地，身体不稳。
    - **likely_cause**: 对垫步脚步控制不熟练，或核心力量不足。
    - **evidence_needed**: 视频分析显示双脚触地时间差大于0.05秒。

## Repair Actions / Practice Tasks
1.  **训练名称：触地即弹训练**
    - **purpose**: 训练利用肌肉千张反射和筋膜弹性，减少深度压缩，实现快速起跳。
    - **setup**: 原地站立，双脚与肩同宽。
    - **dosage**: 10次/组，3组。
    - **drill**: 双脚轻轻跳起，在空中微屈髋膝踝，落地瞬间立即再次跳起，尽量缩短触地时间。想象地面是烫的。
    - **success_metric**: 落地到再次起跳的时间极短，声音清脆，没有明显的下蹲停顿。

2.  **训练名称：同步起球训练**
    - **purpose**: 训练蹬地与起球的同步性，改善上下肢发力脱节。
    - **setup**: 持球，双脚与肩同宽，球置于腰部或胸部高度。
    - **dosage**: 10次/组，3组。
    - **drill**: 做“触地即弹”训练，但在起跳的同时，将球从持球位置快速举到出手点。专注于“蹬地”和“起球”两个动作同时开始。
    - **success_metric**: 起跳和举球动作完全同步，没有先后顺序。投篮发力感觉顺畅。

3.  **训练名称：垫步停球稳定性训练**
    - **purpose**: 训练双脚同时落地，提升垫步投篮的稳定性。
    - **setup**: 从三分线外运球或接球，向篮下移动。
    - **dosage**: 10次/组，3组。
    - **drill**: 在移动中做垫步停球，要求双脚同时落地，并保持身体稳定。可以配合投篮或不投篮。
    - **success_metric**: 双脚触地声音为一声，而不是两声。落地后身体没有明显晃动。

## False Positives
- **误判：深度压缩 = 错误。** 对于力量不足或需要远距离投篮的初学者，深度压缩可能是必要的蓄力方式。该规则主要针对已经有一定基础，希望提升投篮速度和效率的中高级球员。
- **误判：垫步 = 唯一正确脚步。** 视频并未否定一二步（1-2 step）的价值，只是强调垫步在特定场景（如移动投篮）下的优势。不应将垫步视为唯一正确的投篮脚步。
- **误判：触地即弹 = 完全不屈膝。** “浅压缩”不等于“不屈膝”。视频强调的是“减小压缩幅度”，而不是完全取消。运动员仍然需要微屈膝来吸收冲击和储存弹性势能。
