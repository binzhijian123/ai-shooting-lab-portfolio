## Step-by-Step Method
1. **准备姿势（站立）**  
   - 动作：双脚与肩同宽，脚尖微朝前（尽量不外八），身体相对竖直。  
   - 观察点：重心是否在脚掌中部，膝盖是否过度弯曲。  
   - APP检查点：检测膝关节角度是否>150°（避免过度屈膝蓄力）。

2. **蓄力阶段（下蹲）**  
   - 动作：以髋关节为主导向后下方“后坐”，核心和臀部绷紧，膝盖小幅弯曲，脚跟主动抬起。  
   - 观察点：躯干是否保持竖直，膝盖是否过度前移，重心是否落在前脚掌。  
   - APP检查点：检测髋关节向后移动量（>5cm），膝关节角度变化（<30°），脚跟离地高度（>2cm）。

3. **蹬地起跳**  
   - 动作：脚趾主动蹬地，利用弹性爆发力快速伸展髋、膝、踝（三伸展），身体向上而非向前。  
   - 观察点：起跳是否迅速，身体是否前冲，脚跟是否完全离地。  
   - APP检查点：检测起跳瞬间脚趾压力峰值（>体重的1.5倍），躯干前倾角度（<10°）。

4. **上肢衔接**  
   - 动作：在髋、膝、踝充分伸展之后，再衔接上肢发力（手肘回正、手型固定）。  
   - 观察点：是否出现“手快脚慢”（上肢先于下肢发力）。  
   - APP检查点：检测肩关节开始上抬的时间与膝关节完全伸展的时间差（应<50ms）。

5. **出手与跟随**  
   - 动作：身体略向后打开角度，球从头顶上方出手，保持手型固定。  
   - 观察点：投篮弧线是否高，球是否偏向左侧（手型不稳）。  
   - APP检查点：检测出手角度（>50°），球离手时手腕角度（>70°）。

## Common Mistakes
1. **错误：屈膝主导蓄力**  
   - symptom: 膝盖过度弯曲，重心前倾，起跳慢。  
   - likely_cause: 习惯用膝盖弯曲来“蓄力”，而不是用髋关节后坐。  
   - evidence_needed: 侧面视频中膝关节角度<120°，躯干前倾>15°。

2. **错误：全脚掌着地**  
   - symptom: 起跳弹速慢，难以快速蹬地。  
   - likely_cause: 脚跟未抬起，导致无法利用脚趾的弹性爆发力。  
   - evidence_needed: 起跳瞬间脚跟未离地，脚趾压力峰值低。

3. **错误：身体前趴**  
   - symptom: 投篮时身体向前冲，球投不过去或弧线低。  
   - likely_cause: 髋关节未向后打开角度，导致发力方向向前而非向上。  
   - evidence_needed: 起跳后躯干前倾>20°，出手角度<40°。

4. **错误：手快脚慢**  
   - symptom: 上肢先于下肢发力，发力脱节。  
   - likely_cause: 上肢习惯性提前启动，下肢力量未充分传导。  
   - evidence_needed: 肩关节开始上抬的时间早于膝关节完全伸展。

5. **错误：手型不稳**  
   - symptom: 球出手后偏向左侧（右手投篮者）。  
   - likely_cause: 出手后手型未固定，手腕或手指方向偏移。  
   - evidence_needed: 球离手后手腕角度变化>10°，球飞行轨迹偏离目标。

## Repair Actions / Practice Tasks
1. **训练：后坐抬脚跟**  
   - purpose: 培养髋关节主导蓄力、脚跟抬起的习惯。  
   - setup: 原地站立，双手叉腰或持球于胸前。  
   - dosage: 10次/组，3组。  
   - success_metric: 每次下蹲时髋关节明显后移（>5cm），脚跟离地>2cm，膝盖弯曲<30°。

2. **训练：脚趾蹬地**  
   - purpose: 激活脚趾发力，利用弹性爆发力。  
   - setup: 原地站立，脚跟抬起，仅前脚掌着地。  
   - dosage: 快速蹬地跳起10次/组，3组。  
   - success_metric: 起跳迅速（从下蹲到离地<0.2秒），落地时脚跟仍保持离地。

3. **训练：三伸展节奏**  
   - purpose: 建立“髋-膝-踝先伸展，上肢后发力”的时序。  
   - setup: 无球或持球，从下蹲姿势开始。  
   - dosage: 慢速分解练习5次，然后正常速度10次/组，3组。  
   - success_metric: 上肢启动时间晚于膝关节完全伸展（<50ms），无“手快脚慢”现象。

4. **训练：后坐打开角度**  
   - purpose: 纠正身体前趴，学会向后打开髋关节角度。  
   - setup: 面对墙壁或镜子，原地做投篮起跳动作。  
   - dosage: 10次/组，3组。  
   - success_metric: 起跳后躯干保持竖直或略后仰（前倾<10°），髋关节向后伸展>5°。

5. **训练：手型固定**  
   - purpose: 确保出手后手型稳定，不偏移方向。  
   - setup: 近距离投篮（篮下或罚球线），专注于出手后手型。  
   - dosage: 20次/组，3组。  
   - success_metric: 球飞行轨迹笔直（不偏左/右），出手后手腕角度保持>70°。

## False Positives
1. **膝盖弯曲角度小≠错误**：如果用户本身力量强、弹跳好，小幅度屈膝可能正常。该规则主要针对“屈膝过度”导致的发力脱节，而非所有小屈膝。
2. **脚跟离地≠必须**：对于近距离投篮或罚球，脚跟不离地可能更稳定。该规则主要针对跳投时的快速起跳。
3. **身体后仰≠正确**：如果用户刻意“后仰”导致重心不稳，反而可能影响投篮。该规则强调“略往后”以打开角度，而非过度后仰。
4. **手型固定≠僵硬**：出手后手型固定是防止偏移，但不应导致手腕僵硬失去柔和度。
5. **矫正效果因人而异**：视频中90分钟见效，但用户可能需要更长时间练习，不应因短期无改善而否定规则。
