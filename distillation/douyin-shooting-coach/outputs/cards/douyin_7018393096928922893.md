## Step-by-Step Method
1. **接球与沉球（顺势下沉）**
   - 动作：接球时利用球下落惯性，顺势沉球至腰部或更低位置。
   - 观察点：沉球是否流畅，是否与身体下降同步。
   - APP 检查点：检测沉球时髋、膝、踝是否同时弯曲；沉球是否由球下落带动而非手臂主动下压。

2. **点脚垫步（蓄力）**
   - 动作：投篮前做点脚垫步，快速弹起。
   - 观察点：垫步是否快速、有力，是否利用超等长收缩。
   - APP 检查点：检测垫步后起跳时间间隔（应极短）；检测小腿肌肉收缩模式（通过加速度变化）。

3. **起球与起跳（同上同下）**
   - 动作：沉球后，身体向上发力，同时顺势起球至投篮位置。
   - 观察点：起球是否与起跳同步，是否利用身体向上力量。
   - APP 检查点：检测起球开始时间与起跳开始时间是否一致；检测手臂是否主动发力（通过肘关节角度变化率）。

4. **站位与身体对齐**
   - 动作：双脚略宽于肩，左脚正对篮筐，右脚内旋形成内八。
   - 观察点：双脚位置是否稳定，身体是否平衡。
   - APP 检查点：检测双脚间距（应大于肩宽）；检测右脚内旋角度（轻微内八）；检测肩部是否与篮筐平行。

5. **正腕起球与手臂正直**
   - 动作：起球时手腕锁定正对篮球中心线，手肘内收，小臂保持正直。
   - 观察点：手腕是否在起球时已锁定；手肘是否内收；小臂是否垂直。
   - APP 检查点：检测手腕角度（起球时是否稳定）；检测肘关节内收角度（应小于90度）；检测小臂与地面夹角（应接近垂直）。

6. **压腕与出手**
   - 动作：投篮时手掌与手指自然贴合球，压腕幅度约180度，辅助手只做上举稳定。
   - 观察点：压腕是否充分；辅助手是否离开球；球是否后旋。
   - APP 检查点：检测压腕角度（应接近180度）；检测辅助手是否在出手前离开球；检测球旋转速度。

## Common Mistakes
1. **错误：沉球时手臂主动发力**
   - symptom: 沉球动作僵硬，手臂下压而非利用球下落惯性。
   - likely_cause: 用户习惯用手臂控制球，缺乏身体协调。
   - evidence_needed: 视频中检测沉球时肘关节角度变化是否与身体下降同步。

2. **错误：起球与起跳脱节**
   - symptom: 起球先于起跳，或起跳后手臂才举球，导致发力不连贯。
   - likely_cause: 用户未掌握“同上同下”节奏，用手臂力量而非身体力量。
   - evidence_needed: 检测起球开始时间与起跳开始时间的时间差。

3. **错误：压腕不充分**
   - symptom: 出手后球后旋不足，弹道不稳定。
   - likely_cause: 手腕柔韧性不足，或手掌未贴合球。
   - evidence_needed: 检测压腕角度（应接近180度）；检测球旋转速度。

4. **错误：辅助手参与发力**
   - symptom: 出手后辅助手跟随球移动，或球飞行轨迹偏斜。
   - likely_cause: 用户习惯用辅助手纠正方向。
   - evidence_needed: 检测辅助手在出手前是否离开球；检测球飞行轨迹是否直线。

## Repair Actions / Practice Tasks
1. **顺势沉球练习**
   - drill: 接球后，让球自然下落至腰部，同时身体下降，不主动用手臂控制。
   - purpose: 培养利用球下落惯性的习惯。
   - setup: 站立，接球后放松手臂，让球自由下落。
   - dosage: 10次/组，3组。
   - success_metric: 沉球时肘关节角度变化与髋关节角度变化同步。

2. **同上同下节奏练习**
   - drill: 无球起跳举球，强调身体向上时手臂同步上举。
   - purpose: 建立身体与手臂的协调发力。
   - setup: 站立，双手模拟持球，起跳同时举球至投篮位置。
   - dosage: 15次/组，3组。
   - success_metric: 起跳开始时间与举球开始时间差小于0.1秒。

3. **压腕练习**
   - drill: 近距离（1-2米）投篮，专注于手腕完全下压。
   - purpose: 提高压腕幅度和球后旋。
   - setup: 篮下，单手投篮，强调手腕下压至180度。
   - dosage: 20次/组，3组。
   - success_metric: 球后旋明显，弹道稳定。

4. **辅助手稳定练习**
   - drill: 单手投篮，辅助手只做上举稳定，不接触球。
   - purpose: 消除辅助手参与发力的习惯。
   - setup: 站立，单手投篮，辅助手放在身体一侧或上举但不碰球。
   - dosage: 15次/组，3组。
   - success_metric: 出手后辅助手保持静止，球飞行轨迹直线。

## False Positives
- 点脚垫步可能被误判为“踮脚投篮”，但汤普森是快速弹起，而非踮脚后停顿。
- 轻微内八站位可能被误判为“双脚不平行”，但这是为了稳定支撑，非错误。
- 正腕起球可能被误判为“手腕僵硬”，但这是为了锁定手腕，非错误。
- 压腕180度可能被误判为“过度压腕”，但这是汤普森的个人特点，非通用标准。
- 该规则不适用于需要高弧度投篮的用户（如库里风格），因为汤普森压腕幅度大，球弧度可能较低。
