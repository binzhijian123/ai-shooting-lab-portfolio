## Core Claims
1. **高肘位与低肘位的发力方向不同**：高肘位发力方向偏前后，低肘位发力方向偏垂直向上。
   - evidence: explicit（视频中教练明确说“如果你是高手的话，更多是前后”；“低的话，你会往上再，肘位到往上顶这一下”）

   - evidence: explicit（学员说“三分的话，我感觉我的肩膀上肢会比较僵”；教练说“已经没有这个发力角度了，相当于这里。就会更多的腰去带长这个角度去投篮”）
3. **低肘位能更好地衔接下肢力量**：通过降低肘位，使大臂与躯干协同向上，让下肢力量更直接地传导到出手。
   - evidence: explicit（教练说“低肘能衔接起来，你也一样能跳起来”；“你脚的力量向上看到”）
4. **大臂与躯干分离是起球砸头和发力不连贯的原因**：大臂外展（打开）会导致起球路径偏离身体中线，容易砸头，且需要腰部代偿。
   - evidence: explicit（教练说“九哥起来就是大臂收住，你是大臂一下就打开”；“你的大臂跟你的躯干一起走，永远不会打到”）
5. **侧身站位有助于低肘位发力**：侧身时，起球路径可以沿着身体弓线（身体中线）向上，避免外翻。
   - evidence: explicit（教练说“你这样的话就会带一些侧身套，侧身套就沿着身体弓线在走”）
6. **急停时脚往前迈可以创造起球空间，减少挺腰需求**：急停时脚向前，身体重心后移，为起球提供了更多空间，从而不需要挺腰。
   - evidence: explicit（教练说“你急停的时候，其实这个脚看，往前走了，然后你的起球空间就有了，就不用去挺腰了”）

## Step-by-Step Method
1. **准备姿势（沉球）**：
   - 动作：双脚与肩同宽或略宽，屈膝沉球，球置于腰部或大腿位置。
   - 观察点：沉球时，大臂是否贴近躯干，肘部是否自然下垂。
   - APP 检查点：检测沉球时肘部与躯干的夹角（理想：< 30度，避免大臂外展）。

2. **起球（大臂与躯干协同）**：
   - 动作：从沉球位置开始，大臂与躯干作为一个整体向上移动，肘部沿身体中线（弓线）向上推，而不是向外拉。
   - 观察点：起球过程中，大臂是否与躯干保持接触或平行移动，肘部是否指向篮筐方向。
   - APP 检查点：检测起球时大臂与躯干的夹角变化（理想：夹角保持不变或逐渐减小，避免突然增大）；检测肘部轨迹是否在身体中线附近。

3. **蹬地与摆臂衔接**：
   - 动作：在起球的同时，下肢蹬地发力，力量通过躯干传导到手臂，形成“蹬地-摆臂”的连贯链条。
   - 观察点：蹬地时，身体是否向上伸展，而不是向后仰；手臂是否在蹬地力量到达最高点时出手。
   - APP 检查点：检测蹬地开始时间与起球开始时间的同步性（理想：时间差 < 0.1秒）；检测身体重心在垂直方向上的位移曲线（理想：平滑上升，无突然后移）。

4. **出手点（低肘位）**：
   - 动作：当球举到额头或略高于额头位置时，肘部处于较低位置（约与肩同高或略低），小臂向上推球，手腕柔和发力。
   - 观察点：肘部是否低于或等于肩部高度，手腕是否在球离手前保持稳定。
   - APP 检查点：检测出手时肘部高度与肩部高度的比值（理想：≤ 1.0）；检测手腕角度变化（理想：从后屈到前屈的平滑过渡）。

5. **跟随动作**：
   - 动作：球离手后，手臂自然向前上方伸展，手腕放松，手指指向篮筐。
   - 观察点：跟随动作是否自然，手臂是否过度外展或内收。
   - APP 检查点：检测跟随动作结束时手臂与垂直线的夹角（理想：10-20度，避免过大外展）。

## Common Mistakes
1. **错误：大臂外展（高肘位）导致发力不连贯**
   - symptom: 三分线外投篮时，上肢僵硬，感觉下肢力量上不来，需要挺腰代偿。
   - likely_cause: 起球时大臂与躯干分离，肘部向外拉，导致发力方向偏前后，无法有效利用下肢力量。
   - evidence_needed: 侧面视频中，检测起球时大臂与躯干的夹角是否突然增大（> 30度）；正面视频中，检测肘部是否明显偏离身体中线。

2. **错误：挺腰（腰部代偿）**
   - symptom: 投篮时身体后仰，腰部过度弯曲，导致发力不稳定，命中率下降。
   - likely_cause: 起球空间不足（大臂外展或胸椎受限），需要腰部后仰来创造出手空间。
   - evidence_needed: 侧面视频中，检测髋部与肩部的连线是否在出手时向后倾斜（理想：垂直或略向前倾）。

3. **错误：起球砸头**
   - symptom: 起球过程中，球碰到头部或面部，导致出手动作变形。
   - likely_cause: 大臂与躯干分离，起球路径偏离身体中线，需要绕开头部的空间。
   - evidence_needed: 正面或侧面视频中，检测起球时球是否经过头部正前方（理想：球沿身体中线向上，不经过头部）。

## Repair Actions / Practice Tasks
1. **训练：低肘位起球练习**
   - drill: 原地站立，不跳投，只练习从沉球到低肘位起球的动作。双手持球于腰部，屈膝，然后大臂与躯干协同向上，肘部沿身体中线推至额头高度，保持肘部低于或等于肩部。
   - purpose: 建立大臂与躯干协同的肌肉记忆，纠正大臂外展的习惯。
   - setup: 无篮筐，在镜子前或使用手机录制侧面视频。
   - dosage: 每天3组，每组10次，每次动作保持2秒。
   - success_metric: 起球时，大臂与躯干夹角保持稳定（< 30度），肘部轨迹在身体中线附近，无挺腰现象。

2. **训练：急停低肘位跳投**
   - drill: 从运球开始，做急停跳投，重点在急停时脚往前迈，创造起球空间，然后使用低肘位起球和跳投。
   - purpose: 将低肘位起球应用到实战场景，特别是远距离跳投。
   - setup: 在三分线内一步位置开始，运球至三分线外急停跳投。
   - dosage: 每天3组，每组5次，记录命中率。
   - success_metric: 急停后，起球流畅，无挺腰，命中率稳定在50%以上。

3. **训练：侧身站位起球**
   - drill: 采用侧身站位（非射手侧脚在前），练习从沉球到低肘位起球，感受起球路径沿身体弓线（身体中线）向上。
   - purpose: 利用侧身站位优化起球路径，减少大臂外展。
   - setup: 无篮筐，在镜子前或使用手机录制正面视频。
   - dosage: 每天2组，每组10次，交替左右脚在前。
   - success_metric: 起球时，肘部轨迹在身体中线附近，无外翻现象。

## False Positives
- **高肘位不等于错误**：视频中教练指出高肘位在中距离（两分）可能更稳定，问题主要出现在远距离（三分）。因此，不应将高肘位一概视为错误，而应结合距离和用户习惯判断。
- **挺腰不一定全是问题**：在某些后仰跳投或高难度投篮中，挺腰可能是必要的。规则卡主要针对远距离发力不流畅时的代偿性挺腰，而非所有挺腰。
- **急停时脚往前迈可能被误判为错误**：视频中教练指出急停时脚往前迈可以创造起球空间，减少挺腰。因此，脚往前迈本身不是错误，而是需要结合起球动作判断是否有效利用了空间。
