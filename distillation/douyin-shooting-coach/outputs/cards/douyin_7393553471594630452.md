## Step-by-Step Method
1. **测试主视眼**：
   - 动作：双眼看远处一个目标（如计时牌方框），双手交叉做小孔，透过小孔看目标。先闭右眼，再闭左眼，观察目标是否移动。
   - 观察点：闭眼后目标是否偏移？若某只眼闭后目标明显移动，则另一只眼为主视眼。
   - APP检查点：提示用户完成测试，记录主视眼（左/右）。

2. **调整站位（基于主视眼）**：
   - **右眼主视**：右脚略向前（约半个脚掌），左脚在后。髋部内收（内切），不要外翻。肩髋同向，朝向篮筐。
   - **左眼主视**：左脚略向后（或右脚略向前），更多侧身。左肩后收，右肩前送。髋部内收，不要外翻。
   - 观察点：前脚脚尖是否朝前（不要外八），髋部是否内收，肩髋是否同向。
   - APP检查点：检测前脚位置（前后偏移量）、髋部旋转角度、肩髋对齐度。

3. **下蹲与蹬伸**：
   - 动作：屈髋屈膝下蹲，重心稍向后（但不要完全在后脚跟）。快速蹬伸，脚踝展开，跳起离地。
   - 观察点：下蹲深度（不宜过深），重心位置（前脚掌支撑），脚踝是否快速展开。
   - APP检查点：检测下蹲深度（髋膝角度）、重心前后位置、脚踝展开速度。

4. **出手与跟随**：
   - 动作：在上升过程中出手，手（虎口内侧）指向篮筐。肩部向前推，不要向左/右旋转。
   - 观察点：出手方向是否与主视眼对齐，肩部是否旋转，头部是否歪斜。
   - APP检查点：检测出手方向向量、肩部旋转角度、头部倾斜角度。

## Repair Actions / Practice Tasks
1. **主视眼测试练习**
   - drill: 小孔法测试主视眼。
   - purpose: 确定主视眼，为站位调整提供依据。
   - setup: 用户面对墙壁或远处目标，双手交叉做小孔。
   - dosage: 测试3次，确认主视眼。
   - success_metric: 能稳定判断主视眼（左/右）。

2. **站位调整练习（右眼主视）**
   - drill: 右脚前移站位练习。
   - purpose: 使主视眼（右眼）与投篮手（右手）对齐。
   - setup: 原地站立，右脚向前移动半个脚掌，左脚在后。髋部内收，保持肩髋同向。
   - dosage: 10次站位调整，每次保持5秒。
   - success_metric: 能稳定保持右脚前、髋内收的站位。

3. **站位调整练习（左眼主视）**
   - drill: 侧身站位练习。
   - purpose: 使主视眼（左眼）与投篮手（右手）对齐。
   - setup: 原地站立，左脚向后移动，身体侧身。左肩后收，右肩前送。髋部内收。
   - dosage: 10次站位调整，每次保持5秒。
   - success_metric: 能稳定保持侧身、左肩后收的站位。

4. **蹬伸练习**
   - drill: 原地跳起投篮（无球或轻球）。
   - purpose: 改善蹬伸不完整的问题。
   - setup: 原地站立，屈髋屈膝下蹲（浅蹲），快速蹬伸跳起，脚踝展开。
   - dosage: 15次/组，3组。
   - success_metric: 能稳定跳起离地，脚踝快速展开。

5. **重心控制练习**
   - drill: 前脚掌支撑下蹲练习。
   - purpose: 改善重心在后脚跟的问题。
   - setup: 原地站立，重心移至前脚掌，屈髋屈膝下蹲，保持前脚掌支撑。
   - dosage: 10次/组，3组。
   - success_metric: 下蹲过程中重心始终在前脚掌。

## False Positives
- **投篮偏左/偏右不一定都是手眼协调问题**：也可能是肘部外展、手腕发力不均、肩部旋转等原因。应先排除其他常见原因。
- **主视眼测试结果可能受环境干扰**：如果目标不清晰或用户视力问题，测试结果可能不准确。
- **站位调整后方向仍偏**：可能还有其他问题（如出手点、手腕控制），不应仅归因于站位。
- **蹬伸不完整可能由力量不足导致**：如果用户力量不足，强行要求跳起离地可能导致动作变形。
- **站距过宽可能是个体差异**：某些用户（如高个子）可能需要更宽站距以保持平衡，不应一概而论。
